Явная и неявная сетки

  #grid { 
  display: grid;
  grid-template-rows: 60px 60px;    //явная строка
  grid-template-columns: 1fr 1fr;   //явный столбец
  grid-auto-rows: 60px;  // высота строки для неявной сетки
  grid-auto-columns: 60px; // ширина столбца для неявной сетки
  // По умолчанию неявная строка использует размер трека (который основан на контенте) .auto
  grid-gap: 10px;
  }
  #grid > div {
  padding: .5em;
  background: gold;
  text-align: center;                       
  }

grid-auto-flow

Указывает будет ли неявная сетка использовать строки или столбцы

  #grid { 
  display: grid;
  grid-template-rows: 80px 80px;
  grid-template-columns: 1fr 1fr;
  grid-gap: 10px;
  grid-auto-flow: column; //чтобы сделать столбцы неявной сетки регулируемыми, нужно использовать grid-auto-columns: 1fr
  }
